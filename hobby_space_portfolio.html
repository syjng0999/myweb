🌟 "최적화를 설계하고, 데이터로 증명합니다"

안녕하세요! 데이터 속에 잠겨있는 사용자의 목소리(Voice of Customer)를 정량적 수치로 포착하고, 이를 만족시키는 정밀한 금융 서비스를 기획하고 싶은 정소연입니다.

저는 대학에서 산업공학(Industrial Engineering)을 전공하며 복잡한 시스템 내부의 보틀넥(병목현상)을 최소화하고, 데이터 기반으로 프로세스를 최적화(Optimization)하는 논리 구조를 훈련했습니다. 이 학문적 기틀은 한 땀 한 땀 규칙을 엄격하게 설계해야 완벽한 옷감으로 완성되는 '뜨개질의 알고리즘'과 맥이 정확하게 일치합니다.

제가 깊이 몰입한 '슈퍼 마리오 메이커 2' 역시 단순 놀이가 아니라, 사용자가 어떠한 지점에서 낙담하고 흥미를 잃어 이탈하는지 데이터를 기반으로 시뮬레이션하고 최적의 UX 맵 동선을 모델링해보는 첫 기획 실험실이었습니다.

1원 단위의 무결성이 중요한 금융 도메인 기획과 분석에 산업공학의 데이터 기반 의사결정 프레임을 적용하고자 합니다. SSAFY 안에서 실제 서비스의 기획부터 런칭까지 완주하고, 정량적 A/B 테스트를 성공적으로 세팅하여 성장을 주도하는 '그로스 PM / 데이터 분석가'가 되겠습니다.

🤝 함께할 프로젝트의 팀원을 찾고 있어요!
